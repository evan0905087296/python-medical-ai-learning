# AI-based Brain MRI Segmentation Project


## Overview

This project aims to develop a deep learning-based
medical image segmentation pipeline for brain MRI analysis.

The purpose of this project is to understand the workflow
of medical imaging AI, including data preprocessing,
model training, segmentation prediction, and evaluation.


## Motivation

Medical imaging AI has become an important research field
for quantitative image analysis and clinical decision support.

This project serves as my first step toward understanding
AI-assisted medical image analysis.


## Dataset

Planned dataset:

- Medical Segmentation Decathlon (MSD)
- Hippocampus MRI Segmentation


## Objective

Develop an AI model capable of:

1. Reading medical imaging data
2. Performing image preprocessing
3. Training a segmentation model
4. Generating segmentation masks
5. Evaluating model performance
